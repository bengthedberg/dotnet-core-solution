```PS> .\build.ps1```

```PS> .\build.ps1 -Target Build```

Test

```PS> .\build.ps1 -Target Test```

Cleanup Code

```PS> .\build.ps1 -Target CleanUpCode```

Validate Coding Style

```PS> .\build.ps1 -Target Validate```

```PS> .\build.ps1 -Target Dependencies-Analyse```
